# Detective Chronicles 🕵️

A text-based detective mystery game developed in Python.

## Features

* Three unique crime cases
* Interactive investigations
* Multiple suspects and clues
* Detective ranking system
* Score tracking
* Menu-driven gameplay

### Case File #103

## Technologies Used

* Python
* Conditional Statements
* Loops
* Functions
